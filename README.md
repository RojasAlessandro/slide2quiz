# Slide2Quiz

AI-powered platform that converts classroom slides into key concepts and quizzes for student self-study and teacher quiz generation.

## Overview
Slide2Quiz aims to transform passive lecture materials into active learning tools.
The system will accept classroom slides such as PDFs or PowerPoints, extract important concepts, and generate quizzes that can be used by both students and teachers.

## Goals
- Help students revise through automatically generated quizzes
- Help teachers create quiz materials faster
- Turn an AI prototype into a more robust deployable product

## Project Structure
- `docs/` — planning, roadmap, architecture, API draft
- `backend/` — Python backend for slide parsing and quiz generation
- `android/` — Android app for quiz delivery
- `sample-data/` — example files for testing

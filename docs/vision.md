# Vision

## Project Name
Slide2Quiz

## Problem
Students often receive lecture slides but do not have a fast way to turn them into active recall practice.
Teachers also spend time manually converting slide content into quiz materials.

## Solution
Build a system that accepts classroom slides (PDF/PPT), extracts key concepts, and generates quizzes that support:
- student self-study
- teacher quiz preparation

## Target Users
- Students who want practice questions from lecture materials
- Teachers who want draft quizzes generated from slides

## MVP
- Upload PDF
- Extract slide text
- Identify key concepts
- Generate quiz questions
- Return structured quiz output
- Android app displays quizzes for students

## Long-Term Goal
A deployable educational product with:
- stable backend processing
- student accounts and quiz history
- teacher review workflow
- Android app ready for Play Store release
